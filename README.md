# 👟 Loja de Tênis em Python

## 📌 Sobre o Projeto

Este projeto é um sistema de loja de tênis desenvolvido em Python no terminal. O programa permite gerenciar produtos, estoque e vendas de forma simples e organizada.

O sistema foi criado utilizando listas, dicionários, funções, estruturas condicionais e laços de repetição.

---

# ⚙️ Funcionalidades

* Cadastro de tênis
* Exibição de tênis disponíveis
* Busca de tênis por código
* Atualização de estoque
* Sistema de compras
* Relatório de vendas
* Remoção de produtos
* Menu interativo

---

# 🖥️ Tecnologias Utilizadas

* Python 3

---

# ▶️ Como Executar o Projeto

## 1. Instale o Python

Baixe o Python no site oficial:

```bash
https://www.python.org/
```

---

## 2. Baixe o arquivo do projeto

Salve o código com o nome:

```bash
loja_tenis.py
```

---

## 3. Execute o programa

Abra o terminal na pasta do projeto e execute:

```bash
python loja_tenis.py
```
